Gatito War

This is a game made during the summer of 2016.
In this game your main task is to destroy opponents base and keeping your base safe with an army of cat soldiers, cars, tanks and more.
This game is meant to be played by two players on the same computer.
Game is available for web and desktop platforms(Pc, Linux, MacOS).
Desktop versions are patched web versions. We have tested only Linux and Windows versions.

This game is uploaded on Itch.io, all downloads are available for free.
To download this game, go to this website, scroll to the bottom of the page.
Now you can support us and donate some money or just get the game for free.
